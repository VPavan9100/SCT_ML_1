# House Price Prediction using Linear Regression

## Objective
Build a Linear Regression model to predict house prices based on:
- Square Footage
- Number of Bedrooms
- Number of Bathrooms

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

## Dataset
House Price Dataset

## Steps
1. Load dataset
2. Split into training and testing data
3. Train Linear Regression model
4. Predict house prices
5. Evaluate using MSE and R² Score
6. Visualize results

## Results
The model predicts house prices based on selected features and is evaluated using regression metrics.

## Output Visualizations
- Square Footage vs Price
- Actual vs Predicted Prices
- Correlation Heatmap
